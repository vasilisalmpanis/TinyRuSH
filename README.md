---

## How to Run

Clone the repository:
```sh
git clone https://github.com/vasilisalmpanis/TinyRuSH.git && cd TinyRuSH
```

To run the project:
```sh
cargo run
```

To build it:
```sh
cargo build
```

The binary will be located in ```./target/debug/TinyRuSH```

```sh
# Run it manually.
./target/debug/TinyRuSH
```
